# Retail Sales Performance Analysis

## Overview
This project analyzes retail sales performance using Python for data exploration and Power BI for interactive dashboard development. The objective is to identify key trends in revenue, profitability, customer segments, and seasonal patterns.

## Tools Used
- Python
- Pandas
- Data Visualization
- Power BI
- Excel

## Project Structure
retail_sales_data_analysis_python.ipynb → Data cleaning and exploratory analysis

retail_sales_performance_dashboard_powerbi.pbix → Interactive dashboard with KPIs and visual analytics

## Key Metrics
- Total Revenue
- Profit
- Units Sold
- Unique Customers
- Profit Margin
- Average Ticket per Customer

## Dashboard Insights
The dashboard highlights:
- Revenue trends over time
- Performance by product category
- Regional sales distribution
- Seasonal sales behavior
- Customer segment contribution

## Purpose
The goal of this project is to demonstrate end-to-end data analysis, from data preparation and exploration in Python to business intelligence visualization in Power BI.
